# multi-copy 0.1.0<br>
  9 copy and paste slots available as atom functions<br>
### key bindings are:<br>
  copy: ctrl + (1-9)<br>
  paste: alt + (1-9)<br>
<br>

  ### Windows:<br>
   all keybinds work<br>
  ### Mac:<br>
   Slot 3: alt-3<br>
   Slot 8: alt-8<br>
   Slot 9: alt-9  &  ctrl-9<br>
