9 copy and paste slots available as atom functions<br>
### key bindings are:<br>
  copy: ctrl + (1-9)<br>
  paste: alt + (1-9)<br>
<br>
## Keybind problems><br>

  ### Windows:<br>
   all keybinds work<br>
  ### Mac:<br>
   Slot 3: alt-3<br>
   Slot 8: alt-8<br>
   Slot 9: alt-9  &  ctrl-9<br>
