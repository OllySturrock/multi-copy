## General


9 copy and paste slots available as atom functions

### key bindings are:

copy: ctrl + (1-9)

paste: alt + (1-9)

## Keybind problems

From testing on windows and mac atom setups (with nuclide also) - 19/08/18<br>

### Windows:
all keybinds work<br>
  
### Mac:
Slot 3: alt-3
Slot 8: alt-8
Slot 9: alt-9  &  ctrl-9
